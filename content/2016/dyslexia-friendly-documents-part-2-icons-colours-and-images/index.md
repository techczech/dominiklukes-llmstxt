Using visuals and colours
Colours

Colours are a great way to help people distinguish the context. Here are some ways you can use them to improve access.

Use different colours for different sections. If you’re printing a guide book, print each section on different coloured paper.
When you create tables use alternating colours for each row.
In PowerPoint presentations, use different colours for each bullet.

Things to avoid



Try to keep to 2-3 colours per page. Any more than 4-5 colours per document would be too confusing.
Do not use colour as the only way to present information. This will exclude people who may not see all colours.

Images and icons

It is very helpful to include images in your document even if they are not perfect illustrations of the text. They play several roles:

Break up the text into smaller chunks
Serve as anchors when scanning text
Trigger memories of the text

It is particularly useful to attach icons to section headings. It is sometimes difficult to find the right icon but any graphical element will make it easier to navigate.

Slow readers will benefit most from this because they can use the images to quickly jump to the right place in the text.

Even if an image cannot explain the content fully on its own, it can serve as a way of forming memories through associations.

Things to consider



Text over an image makes it harder to process
Too many images can be confusing
Photos are better than drawings unless they have a consistent style. But, photos with people can quickly become dated.
Colourful clipart can be confusing – choose simple icons with 1-2 colours
Images of text cannot be read through text to speech. If you need an image of text, include the same information in plain text, as well.
Don’t be afraid to crop images. It is often more useful to present just part of an image than a tiny image.

Flow charts and Info Graphics

It is often beneficial to present information in a graphical way instead of as a block of text.

We already had a look at Mind Maps and Outlines as a way to structure complex documents.

Another way to present text with a lot of factual information is to create an info graphic. An info graphic takes the key facts and adds images and graphs to illustrate them.

If you are presenting instructions, create a flow chart. It will not only be easier to read but it will also force you to consider the key steps.

Things to avoid



Many Info Graphics are just too busy and complicated to be helpful. Keep them simple.
Flowcharts that have too many elements can be great for engineers but not for simplifying the text.
Always present the information in the flow chart and info graphic as plain text to make sure people with vision problems can access it using text to speech.

Resources and tools

Most people worry that using graphics is too time consuming, too difficult or too expensive. But there are now a large number of free or cheap sources of graphics and great free or cheap tools for working with them.

Sources of images

Last year, we had a look at many free online sources of images. My favourites are:

Pixabay for free photos. All the photos in this post are from Pixabay
The Noun Project for free icons in a consistent, appealing style.
Open Clipart for free clip art.

If you would like to use icons as part of your text, you can also install the free Font Awesome. Just be careful not to send the document to someone who does not have the font.

Online tools

There are many online tools for creating flow charts, mind maps and info graphics. Most of them have a free tier. I recommend:

Lucid Chart or Draw.io for flow charts or diagrams
Piktochart for info graphics. I used Piktochart to create the simple info graphic in this document in about 5 minutes.

Software
Flow Charts

The easiest way to create flow charts is to use the SmartArt feature in Microsoft Word. I used it to create the two flow charts here.

But if you want something more complicated, you can download some free diagramming software. My favourite is yEd.

You can also use MindMapping software such as XMind to create flow charts.

Image editing

You can do most of the basic image editing inside your word processor. Microsoft Word is particularly powerful. It can:

Crop and resize
Change colour
Remove background

But if you want to do more editing, you have many free options. I like to use Paint.NET which has the right balance of usability and power for me. If you don’t want to install software, you can use the free online image editor Pixlr.

I also use a lot of screen shots. My favourite tool for that is Greenshot which also has basic editing features.
